# Multivariable Calculus Solver

A modern symbolic mathematics application built with Python, Tkinter and SymPy.

InfinityCalc provides symbolic integration, multiple integral evaluation, and an intuitive graphical user interface for mathematical computation.

## Features

- Symbolic Integration
- Integration by Parts
- Substitution Method
- Double Integrals
- Error Handling
- Mathematical Formatting
- Modern GUI

## Built With

- Python
- Tkinter
- SymPy
